## Main Features

- `macOS Downloader PRO`
  - Download macOS packages through guided terminal flow.
- `System Updater`
  - Update using detected installers from `/Applications`.
- `Media Creator`
  - Build bootable media with `createinstallmedia`.
- `Little MountEFI`
  - Scan and mount EFI partitions quickly.
- `System Info`
  - Hardware + OS diagnostics in one view.
- `Appearance Settings`
  - Light, Dark, and Matrix themes.
